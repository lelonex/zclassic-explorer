================================================================================
    INSTALLATION QUICK REFERENCE
================================================================================

Prerequisites:
   • Elixir 1.7+
   • Erlang/OTP 22+
   • Node.js 14+
   • Zclassic node (zclassicd) with indexes enabled
   • PostgreSQL (optional, can use in-memory cache)

Setup Steps:
   1. Configure zclassicd with required indexes (see docs/NODE_SETUP.md)
   2. Clone repository: git clone https://github.com/lelonex/zclassic-explorer.git
   3. Install dependencies: mix deps.get && cd assets && npm install
   4. Configure environment: cp .env.example .env && edit .env
   5. Start explorer: source .env && mix phx.server
   6. Access: http://localhost:4000

Production Deployment:
   See docs/DEPLOYMENT.md for complete production setup including:
   • Nginx reverse proxy
   • SSL/TLS configuration
   • Systemd services
   • Monitoring setup
   • Backup strategy

================================================================================
    KNOWN LIMITATIONS
================================================================================

• PostgreSQL is optional but recommended for production
• Full blockchain sync required (can take 6-24 hours)
• Indexes must be enabled BEFORE sync (reindexing is time-consuming)
• Remote node connection possible but not recommended (latency issues)
• Shielded transaction details may be limited (privacy by design)
